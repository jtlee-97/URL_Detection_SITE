[SQL 설정방법]

1. 먼저 localhost로 접근하여 계정을 만든다. (user1/12345)

2. 데이터베이스 sample을 만든다.

3. 첨부된 sql 파일 3개 (board/members/notice)를 가져온다.

4. localhost 권한으로 사용자 user1의 sample 데이터베이스에 대한 권한 설정을 진행한다.

5. config.inc.php의 파일을 첨부된 파일과 같이 수정한다.

6. user1으로 접속 후 사용.


** 데이터베이스 오류로 접근 및 삭제 등이 안되는 경우는 localhost에서 mysql 데이터 베이스를 전체 선택 후 복구 시키면 된다.**
